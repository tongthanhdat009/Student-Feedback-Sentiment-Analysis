# kaggle

## Purpose
Reusable React components for the Kaggle Notebook Manager UI.

## Key Files
| File | Description |
|------|-------------|
| `AccountTable.tsx` | Table listing accounts with test/delete actions |
| `AccountCreateModal.tsx` | Create account form modal |
| `JobTable.tsx` | Table listing jobs with download/open actions |
| `StatusBadge.tsx` | Colored status pill (completed/failed/running/pending) |
| `NotebookTriggerModal.tsx` | Notebook trigger form |

## For AI Agents
- Components receive data via props and callbacks
- API calls via kaggleApi, not directly via fetch
- CSS via Tailwind utility classes
